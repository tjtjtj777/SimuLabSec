# 🚀 SimuLabSec - 半导体工艺仿真平台（增强版）

## 📌 项目简介

SimuLabSec 是一个面向半导体工艺优化的仿真平台，支持对晶圆（Wafer）在光刻过程中的对准（Overlay）进行模拟分析。

用户可以通过配置不同参数（如波长、对准策略、工艺条件等），生成仿真结果并进行可视化分析，从而辅助工艺优化决策。

该项目为 SimuLab 的增强版本（Sec = Secure / Scalable），在原有基础上进一步优化了：

- 系统性能
- 并发计算能力
- 架构可扩展性

在我的半导体仿真项目中，单次计算需要处理约 20**万级别的 wafer 数据点**，涉及多维参数计算（套刻误差、对准策略等），属于典型的**高计算 + 大数据量可视化场景**。

![image.png](65ec308a-66f2-463a-971a-57209f62b145.png)

支持**多图并行计算**，单次请求需要处理约 **80万级数据点（4 × 20万）**，并同时生成4张统计图。

## 🧠 核心功能

### 🔬 1. 仿真计算

- 支持 wafer 数据生成
- 支持 overlay 套刻误差计算
- 多参数组合仿真（波长、级次等）

### ⚙️ 2. 并发计算优化

- 使用线程池 + 并行计算模型
- 支持任务级并发 + 任务内并行（两层并发）
- 提升计算效率

### 📊 3. 数据可视化

- Wafer 色温图（Heatmap）
- 折线图 / 散点图 / 直方图
- 表格数据展示
